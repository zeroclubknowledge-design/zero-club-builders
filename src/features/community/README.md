# Community

Owns social media behavior: feed, posts, comments, likes, shares, bookmarks, stories, groups, events, messaging, media, and notifications.

Current route entry points include:

- `src/routes/app.index.tsx`
- `src/routes/app.compose.tsx`
- `src/routes/app.post.$id.tsx`
- `src/routes/app.chat.*.tsx`
- `src/routes/app.clubs.*.tsx`
- `src/routes/app.notifications.tsx`
- `src/routes/app.bookmarks.tsx`
